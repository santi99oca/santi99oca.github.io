    Aplicando un poco de CSS

    Marzo 19, 2021

    Width, height y overflow

    Width sirve para definir que tan ancho es un bloque ya sea por porcentaje o por pixeles
    Height sirve como Width pero para el alto
    Over flow sirve para darles propiedades cuando el cuadro es mas pequeño que el tamaño de contenido que deseamos.

    Border, Margin y Padding

    sirve para modelar las cajas que es como se representa CSS en el navegador
    Border: puede escribir sus propiedades de forma abreviada como {width, style, color} asi border: 5px solid red ;

    Position

    En Css con el atributo position, podemos darle valores como relative, absolute o fixed.
    Absolute sirve para darle un valor fijo a una caja, especificando la posicion absoluta, con absolute pasan cosas y es que se puede sobreponer, por lo que hay que usarlo con cuidado, y especificar su z-index
